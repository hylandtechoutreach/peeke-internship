Yellow Group

Welcome to the Yellow Group Page! Here you will find some important links and other resources.

Project Information

The goal of this project is to build a web application that allows caregivers to find universal changing tables for their dependents. The working title for the app is UCT Locator.

Google Drive Folder

Access the project Drive folder here. It contains some important details about the project.

GitHub Repository

Access the GitHub repository for the UCT Locator project here. This repository has both the front-end and the back-end. Additionally, issue tracking is available here.

Technologies

The application uses a number of technologies. Here are some of the most important ones:

Back-end

  • NodeJS + Express (Web Server)
  • MongoDB (Database)

Front-end

  • React
